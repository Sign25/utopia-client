# Utopia Client

Клиент-приложение для распределённой эволюции Мира NeuroCore.
Запускает локальную колонию у себя на ПК, шлёт статистику на divisci.com.

## Установка (Windows)

1. Скачайте `utopia-client.zip` со страницы Кабинет (`divisci.com/me`).
2. Распакуйте в любую папку.
3. Запустите `install.bat`. Скрипт установит Python 3.12 embedded,
   создаст venv, поставит зависимости (PyTorch CUDA при наличии).
4. После установки появится команда:
   ```
   utopia-client.bat
   ```
   Используйте её для запуска клиента.

## Установка (Linux / macOS)

```bash
bash install.sh
```

## Конфигурация

При первом запуске клиент попросит:

- **Push-токен** — берётся в Кабинете (`divisci.com/me`).
- **Имя колонии** — короткое уникальное имя, например `home-3060ti`.

Конфиг сохраняется в `~/.utopia-client/config.json`.

## Состояние

Веса организмов хранятся локально в `~/.utopia-client/colonies/<name>/`.
На сервер уходит только сводка (n_alive, лучший фитнес, метрики Phase 1-6).

## Команды

- `utopia-client run` — запустить heartbeat и подключиться к VPS.
- `utopia-client benchmark` — измерить производительность ПК.
- `utopia-client config` — показать текущий конфиг.

## Требования

- Python 3.12 (или будет установлен installer'ом)
- 4 ГБ RAM минимум
- NVIDIA GPU (опционально, для CUDA)
